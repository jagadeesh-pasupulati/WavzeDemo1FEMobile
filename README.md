# Wavze Frontend

Angular frontend application for Wavze Lead Management System.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

The application will be available at `http://localhost:4200`

## Build

```bash
npm run build
```

## Features

- Dashboard with statistics and customer records
- Customer list with search and filtering
- Add new customer functionality
- Customer details view
- AI Assistant popup
- Responsive design with Tailwind CSS
- PrimeNG components for UI

## Structure

- `src/app/components/` - Angular components
- `src/app/services/` - Angular services for API communication
- `src/app/models/` - TypeScript interfaces/models
- `src/app/routes.ts` - Application routing configuration
